# cell-calculation-automation

Project made to help poor biologists who feel pain (emotionally, not physically) when counting cells in Goryaev's chamber.

## What is Goryaev's Chamber?
Goryaev's chamber (also known as a hemocytometer or Goryaev counting chamber) is a special microscope slide used in laboratory medicine for manual cell counting. It has a precisely etched grid that allows for accurate counting of cells in biological samples, particularly blood cells. This device has been a fundamental tool in hematology and cell biology for decades.

## Current Capabilities
Right now, the project focuses on **counting normal blood cells** - the most common and average task performed in clinical laboratories. This automation helps reduce manual counting errors and speeds up the analysis process.
